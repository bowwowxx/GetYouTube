# GetYouTube
捉取youtube影片  
**go run main.go Youtube id**  

ex:  
**go run main.go 1GJA4c_SBT0**  

or  
直接使用bin資料匣編譯完的執行檔  
mac:GetYouTube 1GJA4c_SBT0  
win32:GetYouTube.exe 1GJA4c_SBT0  
win64:GetYouTube64.exe 1GJA4c_SBT0
